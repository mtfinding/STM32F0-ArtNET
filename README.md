# STM32F0-ArtNET
ArtNET evaluation board with STM32F030 and ENC28J60 for WS28xx pixel evaluation designs.
  
Features:  
- STM32F030C8 MCU
- ENC28J60 ethernet controller IC
- SWD interface
- RJ45
- 85mm x 45mm
